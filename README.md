# **Renderoka**

### A 3D Game Engine Built with Vulkan

Renderoka is currently in its initial stage. I plan to use GLFW for Window creation and use **the** Vulkan API to build its initial version, which has only a triangle drawn in a window.

Currently, I'm following the [Vulkan Tutorial](https://vulkan-tutorial.com/). The code in this repository is based on their [open-source work](https://github.com/Overv/VulkanTutorial.git) as I embark on my journey to learn the Vulkan API.
